# Gym Tracker App

A simple, fast web app for tracking your workouts on your iPhone.

## Features

- **Workout Entry**: All 15 exercises visible at once with previous workout data shown
- **Local Storage**: All data saved on your device
- **History View**: See all past workouts with dates and times
- **Progress Charts**: Visual charts showing weight and rep progression for each exercise

## How to Use

### Option 1: Open Locally
1. Simply open `index.html` in Safari on your iPhone
2. For easy access, tap the Share button and select "Add to Home Screen"
3. The app will now appear on your home screen like a native app

### Option 2: Host Online (Recommended)
Host the `index.html` file on any web server or service:
- **GitHub Pages**: Upload to a GitHub repo and enable Pages
- **Netlify**: Drag and drop the file to netlify.com/drop
- **Vercel**: Deploy with Vercel CLI or web interface
- **Any web hosting**: Just upload the single HTML file

Then access it from your iPhone's Safari browser.

## Tips

- You don't need to fill in all exercises every workout - only fill in what you did
- The app shows your previous weight and reps for easy reference
- Data persists in your browser's local storage
- Works offline once loaded
- Optimized for iPhone 15 Pro screen size
